Vr Simulator

Projeto: Simulação de Óculos VR Web

Este projeto é uma página web que emula o funcionamento de um óculos VR utilizando HTML, CSS e JavaScript. A interface consiste em um container com duas divisões circulares que simulam as lentes do óculos. Essas lentes capturam e projetam o conteúdo de outra aba do navegador em tempo real. O funcionamento é simples e intuitivo: Botão de Início: Inicia a captura da imagem ou vídeo da aba selecionada, utilizando a API de captura de tela do navegador. Botão de Encerramento: Finaliza a captura e encerra a projeção nas lentes simuladas. Este projeto demonstra a capacidade de integrar recursos de captura de mídia e manipulação de elementos visuais, criando uma experiência interativa que reflete os conceitos de um dispositivo de realidade virtual.
