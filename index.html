Vr Simulator

Projeto: Simulação de Óculos VR Web

Este projeto é uma página web que emula o funcionamento de um óculos VR utilizando HTML, CSS e JavaScript. A interface consiste em um container com duas divisões circulares que simulam as lentes do óculos. Essas lentes capturam e projetam o conteúdo de outra aba do navegador em tempo real. O funcionamento é simples e intuitivo: Botão de Início: Inicia a captura da imagem ou vídeo da aba selecionada, utilizando a API de captura de tela do navegador. Botão de Encerramento: Finaliza a captura e encerra a projeção nas lentes simuladas. Este projeto demonstra a capacidade de integrar recursos de captura de mídia e manipulação de elementos visuais, criando uma experiência interativa que reflete os conceitos de um dispositivo de realidade virtual.

Página Web API

Página Web para E-commerce com CRUD de Produtos

Este projeto é uma página web desenvolvida para gerenciar produtos em um sistema de e-commerce. A interface é intuitiva e eficiente, permitindo que o usuário realize operações de CRUD (Create, Read, Update, Delete) diretamente com os dados armazenados no banco de dados, utilizando uma API para integração. Funcionalidades principais: Cadastro de Produtos: Permite adicionar novos produtos ao sistema, fornecendo informações como nome, descrição, preço e quantidade. Alteração de Produtos: Facilita a edição de dados de produtos já cadastrados, garantindo que as informações estejam sempre atualizadas. Exclusão de Produtos: Permite remover produtos obsoletos ou incorretos diretamente do banco de dados. Visualização de Produtos: Exibe os produtos cadastrados em uma lista organizada, com todas as informações relevantes. Design e Navegação: Sidebar Dinâmica: Uma barra lateral foi adicionada para facilitar a navegação, oferecendo opções de funcionalidades futuras, como gerenciamento de categorias, relatórios e configurações do usuário. Integração com API: Toda comunicação com o banco de dados é feita por meio de uma API, garantindo maior segurança e escalabilidade no acesso e manipulação dos dados.
